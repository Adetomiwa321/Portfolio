Phone Contracts

This web application was created using javascript to create a list of requirements that needed to be fulfilled before displaying a match for the product required by the user. The user input their preference in phone based on (brand, monthly cost, minutes etc.) and based on these preferences, the application would recommend suitable phones and plans (phone contracts).

    USAGE
  1. Enter a brand
  2. Enter a minimum data allowance
  3. Enter a minimum number of talk minutes
  4. Enter a minimum text allowance
  5. Enter a minimum monthly cost
  6. Display of plan(s) that match your preferences
